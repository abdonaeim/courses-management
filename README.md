# Courses Management Plugin

![Banner](assets/images/Banner.jpg)

## Overview
Plugin to create and manage courses, sessions, and student enrollments from within WordPress.

## Features
- Create Courses
- Add Sessions for each course
- Track Student Attendance
- Assign Courses to Students
- Dashboard with Reports

## Screenshots
![Dashboard](assets/images/dashboard.png)
![Sessions](assets/images/addnewcourse.png)

## Installation
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through WordPress Admin
3. Go to Dashboard > Courses Management

## Usage
After activation, use the Dashboard to manage courses, sessions, and students. Full reports are available under the Dashboard > Reports tab.
