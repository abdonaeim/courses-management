# Courses Management Plugin

![Banner](./assets/images/Banner.jpg)

## Overview
**Courses Management Plugin** is a powerful WordPress plugin designed to create, manage, and track courses, sessions, and student enrollments directly from your WordPress dashboard.  
It provides an intuitive admin interface and detailed reports to help you manage your educational content efficiently.

---

## Features
- **Create Courses:** Easily add and organize courses.
- **Add Sessions:** Create multiple sessions within each course.
- **Track Student Attendance:** Mark attendance for each session.
- **Assign Courses to Students:** Manage student enrollments seamlessly.
- **Dashboard & Reports:** Comprehensive dashboard with performance and attendance reports.

## Installation
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through WordPress Admin
3. Go to **Dashboard > Courses Management**

---

## Usage
After activation:  
- Use the Dashboard to manage courses, sessions, and students.  
- View detailed reports under **Dashboard > Reports**.  
- Track student attendance and course completion.
